# 🏠 첫페이지 재설계 플랜 — Company Catcher v2

## 🎯 핵심 컨셉

> **"한 화면에서 오늘의 모든 기사화 가능한 단서·기사·트렌드를 즉시 파악"**

기자가 출근해서 첫 페이지만 보면:
- 오늘 출고할 단신 5건
- 회사 IR 통화해야 할 심층 5건
- 산업 트렌드 1~2건
- 정보 격차 큰 신규 발굴 종목 3건

---

## 📐 페이지 구조 (5섹션)

### 🚨 Section 1 — 오늘의 핫 단서 (Hero)
**목적:** 가장 중요한 5건 즉시 노출

```
┌─────────────────────────────────────────────────┐
│  🔥 오늘의 핫 단서   (severity 5 + 새로움 90+)    │
├─────────────────────────────────────────────────┤
│  🚨 LG이노텍, Apple 매출 의존도 81%              │
│     [DEPENDENCE] [정보격차 ⭐]                   │
│                                                 │
│  🆕 글로본, 대규모 신사업 진출 — 새로움 98점       │
│     [NEW] [v3로 작성] [심층 가능]                │
│                                                 │
│  📰 한국기업평가, ESG 사업 진출 (SAFE 80점)        │
│     [STRAIGHT 즉시 출고]  [기사 보기]             │
│                                                 │
│  ... (5건)                                      │
└─────────────────────────────────────────────────┘
```

### 📊 Section 2 — 오늘의 통계 카드 (KPI)
**4개 카드 한 줄:**

| 🆕 신규 단서 | 📰 출고 가능 | 🎯 심층 대상 | 🌍 정보 격차 |
|---|---|---|---|
| 9건 | 55건 | 116건 | 2,414개사 |
| (24h 내) | (검증 LOW+) | (v3 + 외부자료) | (🚨 등급) |

각 카드 클릭 → 해당 페이지 이동

## 🗂 사이드 네비게이션 재구성

기존 nav 메뉴를 **콘셉별로 그룹화:**

```
🏠 홈 (대시보드)
─────────────
📰 기사
  ├ 즉시 단신 (STRAIGHT)
  ├ 심층 기사 (REPORTER)
  └ 모든 기사

🚨 단서
  ├ 일반 단서 (story_leads)
  ├ 공급망 단서 (supply_chain_leads)
  └ 수시공시 단서 (event_leads)

🕸 공급망
  ├ 허브 분석
  ├ 의존도 리스크
  ├ 클러스터
  └ 산업종합

🏢 기업
  ├ 종목 검색
  ├ 정보격차 순위
  └ 비교분석

📊 분석
  ├ 새로움 검증
  ├ 변화 유형
  └ 검증 점수
```

## 🎨 디자인 원칙

### 1. 정보 밀도 ↑
- 5섹션 모두 한 화면에 (스크롤 1번)
- 각 카드는 핵심 3정보만 (제목·점수·바로가기)

### 2. 색상 코딩
| 영역 | 컬러 |
|---|---|
| 즉시 단신 (STRAIGHT) | 🟢 초록 (안전) |
| 심층 (REPORTER) | 🟣 보라 (분석) |
| 공급망 | 🔵 파랑 (네트워크) |
| 리스크 | 🔴 빨강 (경보) |
| 정보 격차 | 🟡 노랑 (주목) |

### 3. 우선순위 표시
- 좌상단부터 중요도 순
- severity 5는 오른쪽 끝에 배지
- 새로움 70+ 단서는 ✨ 마크

## 🔧 구현 단계

| Phase | 작업 | 시간 |
|---|---|---:|
| **HP-1** | API 통합 — `/api/dashboard` 신설 | 2h |

| **HP-3** | 실시간 수시공시 피드 (자동 갱신) | 1.5h |
| **HP-4** | 사이드 네비 재구성 | 1.5h |
| **HP-5** | 색상·아이콘·타이포 통일 | 1h |
| **HP-6** | 모바일 반응형 (480px+) | 2h |

**총 11시간**
